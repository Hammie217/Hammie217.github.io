EEE6225 Systems Design

Tim's Challenge
Your mission should you choose to accept it:
  • Write a testbench which will perform AES encryption using VHDL. This only has to execute using Modelsim and does not need to be synthesisable. This is essentially a complete behavioral model for the AES.
  • It should be capable of being executed in the testbench by a statement such as:
    • ciphertext <= AES(plaintext,key);
    • where the inputs and outputs are std_logic_vector(127 downto 0)
  • The challenge is that the entrire description for the AES is to be done in less than 100 LINES of VHDL
    • A line must be less than 100 characters long
    • Each line contains a maximum of one statement (statements end with semicolons)
    • There must be a blank line between each procedure or function you define - this is also a hint ;-)
    • The chosen symbol names and structure must be meaningful and readable.
    • You need only describe encryption for a 128-bit key.
