C Basics

The barebones of every C should start looking like this:
#include <stdio.h>
        
        int main (void){
        
        return (0);
        }
whether your ide creates this for you or if you have to code this yourself, everything begins here!

What each line does:

#include <stdio.h>- This tells the code which library (a list of functions) should be added to the program, this one is the most basic and most commonly used and will always be used in command line projects.

int main (void){- this just defines the main (the area where your code will always start) for people who have programmed before you may notice that main is an int whereas other languages are voids, main will still work as a void but it is globally accepted that main should be an int and at the end should return 0.

return (0);-this is just the end of the program, as when your program is run it is usually through the command line and should return this 0 value.

{ }- The brackets from main to return define the area of code classed as this main.

;- Almost every line of code in C is ended with ;, these are generally only not used for loops or checking statements such as while and if statements.
