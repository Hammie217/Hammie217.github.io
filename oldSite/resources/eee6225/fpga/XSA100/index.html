EEE6225 Systems Design

Xess XSA-100 Board

Features:

  • XC2S100 FPGA
  • XC9572XL CPLD
  • 16 MByte SDRAM
  • 256 KByte Flash
  • 100 MHz programmable oscillator
  • Parallel port
  • Keyboard/mouse PS/2 port
  • 64-color VGA port
  • 7-segment LED
  • 1 pushbutton
  • 4 DIP switches
  • 84-pin prototyping interface (53 I/O pins)
  • 5V-tolerant I/O
  • 9V DC power jack
  • 5V / 3.3V / 2.5V regulators
