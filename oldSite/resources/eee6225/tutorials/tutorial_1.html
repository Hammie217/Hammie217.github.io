EEE6225 Systems Design

Tutorial 1: Introduction to VHDL and Xilinx ISE
Introduction
You will be asked to construct a simple logic circuit using a Hardware Description Language (HDL) and then use this information to implement and program a Field Programmable Gate Array (FPGA) on a Demonstration PCB. You will supply logic level signals to the inputs and record the value of the outputs to ascertain the logic function.
Outcomes
There are two outcomes associated with this exercise:
  • A demonstration of the use of HDLs
  • Gaining familiarity with the Xilinx ISE (Integrated Synthesis Environment)
Equipment
The equipment used will be:
  • Digilent Spartan III board
  • Power Supply Adaptor for above
  • PC
  • Xilinx ISE 14.x
Activities
After logging in to one of the PCs, find the Project Navigator icon on the desktop and run this application.

Alternatively this can be done from the windows menu:

Start > All Programs > Xilinx Design Tools > ISE Design Suite 14.x > ISE Design Tools > xx-bit Project Navigator
