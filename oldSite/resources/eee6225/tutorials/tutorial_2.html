EEE6225 Systems Design

Tutorial 2: Linear Feedback Shift Register
Introduction

You will be asked to construct a simple, sequential logic circuit - a linear feedback shift register (LFSR) - using a Hardware Description Language (HDL) and then use this information to implement and program a Field Programmable Gate Array (FPGA) on a Demonstration PCB. In this case, you will need to simulate the behaviour of the circuit block (within ISim) to show that it is functioning properly and then connect this circuit block through to an oscillator block to make it function on the FPGA at a speed where a human can view its activity.
Outcomes
There are three outcomes associated with this exercise:
  • A further demonstration of the use of HDLs
  • Gaining more familiarity with the Xilinx ISE (Integrated Synthesis Environment)
  • An introduction to the ISim simulator
Equipment
The equipment used will be:
  • XSA100 FPGA prototyping board or Digilent Spartan 3 board
  • Power Supply Adaptor for above
  • PC
  • Xilinx ISE 14.x which includes the ISim simulator
Activities

Find the Xilinx ISE Project Navigator on the menu tree or the desktop and run this application.
