EEE6225 Systems Design

Files required for Ethernet Link

The following files need to be ADDED to the project, by using the “Add Copy of Source” command. You can get them here.

  • testudp.vhd - This is the top level entity of your design, which is called test_udp. You will need to modify this file to include your design as a component in this entity.

  • globalconstants.VHD - This is a VHDL package that contains parameters for the Ethernet Core, which includes the IP address of the PC that the FPGA will be communicating with. DO NOT change anything in this file.

  • udppins.ucf - This is called a constraint file, which specifies the I/O pin mapping of test_udp to the FPGA. DO NOT modify this file.

  • bridge_dpram_512x8.xco - This is dual port ram that is generated using the generator.
Some other files need to be COPIED to the project directory. You can access them here.
